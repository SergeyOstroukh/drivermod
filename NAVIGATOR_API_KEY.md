# Настройка ключа API для Яндекс.Навигатора

## Проблема

Яндекс.Навигатор имеет лимит на количество вызовов из сторонних приложений:
- **Без ключа API**: максимум 5 вызовов в сутки на одно устройство
- После превышения лимита переходы открывают веб-версию вместо приложения

## Решение

### 1. Получить ключ API

1. Перейдите на страницу: https://yandex.ru/set/lp/navigatorb2b
2. Заполните форму заявки на получение ключа
3. Дождитесь получения ключа от Яндекс

### 2. Добавить ключ в код

Откройте файл `js/app.js` и найдите строку:

```javascript
const YANDEX_NAVIGATOR_API_KEY = ""; // Вставьте ваш ключ здесь
```

Вставьте полученный ключ между кавычками:

```javascript
const YANDEX_NAVIGATOR_API_KEY = "ваш_ключ_здесь";
```

### 3. Проверка работы

После добавления ключа:
- ✅ Все вызовы навигатора будут работать без лимитов
- ✅ Предупреждение о лимитах исчезнет из интерфейса
- ✅ Маршруты будут открываться в приложении навигатора

## Альтернатива

Если не хотите получать ключ:
- Используйте Яндекс.Карты (без лимитов)
- Или ограничьте использование навигатора до 5 раз в сутки

## Дополнительная информация

- Документация: https://yandex.ru/dev/navigator/doc/ru/
- Поддержка: через форму на сайте Яндекс
